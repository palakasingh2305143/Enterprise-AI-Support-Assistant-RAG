 # Enterprise AI Support Assistant

An enterprise-grade conversational AI assistant built using OpenAI GPT, LangChain, ChromaDB, and Retrieval-Augmented Generation (RAG).

## Features

- Document Question Answering
- Conversational AI
- Semantic Search
- Vector Database Retrieval
- Customer Support Automation
- Enterprise Knowledge Management

## Technology Stack

- Python
- OpenAI GPT
- LangChain
- ChromaDB
- Streamlit
- Hugging Face Embeddings

## Workflow

1. Upload PDF knowledge base
2. Generate embeddings
3. Store vectors in ChromaDB
4. Retrieve relevant context
5. Generate responses using GPT

- Multi-Agent AI Workflows
- CRM Integration
